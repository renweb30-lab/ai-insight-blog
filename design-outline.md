# ブログサイト設計アウトライン

## 1. サイトのゴール
- Notionで管理していた記事を、視認性と回遊性の高い独自ブログとして再構築する。
- AI初心者〜中級者がストレスなく学べる情報整理と導線を整備する。
- 継続的な記事追加・更新がしやすい情報設計とコンポーネント設計を用意する。

## 2. ターゲット
- AIリテラシーを高めたい個人投資家・ビジネスマン
- 最新AIニュースやツール活用法を探している情報感度の高い社会人
- AI投資に興味がある初心者層

## 3. 情報設計 / IA
### グローバルナビゲーション
1. Home（トップ）
2. Categories（カテゴリ一覧）
3. Archive（アーカイブ）
4. Profile（プロフィール）
5. External Links（Instagram / ココナラ / Threads）

### コンテンツ分類
- AI株情報
- AI入門
- AI関連ニュース
- AIツール紹介

### 推奨タグ構造
- メインタグ：`<main>` 内に記事一覧・セクションを配置
- セクション：おすすめ記事、最新記事、カテゴリナビ
- 補助要素：サイド情報はモバイルではフッターに、デスクトップではサイド

## 4. レイアウト & UI
- モバイルファースト（幅360px〜）で設計し、`max-width: 1200px`のコンテナを中心に配置
- ヘッダー：ロゴ・ナビ・ハンバーガーメニュー
- トップヒーロー：キャッチコピー＋最新記事へのショートカット
- カードUI：余白を活かしサムネイル無しのテキスト中心、カテゴリバッジをカラー表現
- カラー：
  - ベース：`#0F172A`（濃紺）
  - アクセント：`#38BDF8`（シアン）
  - 背景：`#F8FAFC`
  - テキスト：`#0B1120`
- タイポグラフィ：`"Inter", "Noto Sans JP", sans-serif`
- アイコン：Font Awesome、外部リンクアイコンに利用

## 5. ページ別概要
### 5.1 トップページ
- ヒーローセクション（サイト説明 / CTA）
- おすすめ記事（3件まで固定表示）
- 最新記事一覧（ページング or "もっと見る" ボタン）
- カテゴリショートカット（バッジ）

### 5.2 カテゴリページ
- カテゴリ名＋説明
- 当該カテゴリの記事カード一覧（新着順）
- カテゴリ内検索バー（将来的に拡張可能なプレースホルダー）

### 5.3 アーカイブページ
- 年月別アコーディオン or グリッド表示
- 月別件数表示
- 日付順（新→旧）

### 5.4 プロフィールページ
- 運営者情報（写真URL差し替え容易に）
- 経歴ハイライト、運用方針
- SNS / サービスリンク

## 6. コンポーネント
- `Header`：ロゴ、ナビ、モバイルメニュー
- `ArticleCard`：タイトル、日付、カテゴリ、要約
- `CategoryBadge`：カテゴリ識別カラー
- `Footer`：コピーライト、SNSリンク
- `Pagination`（JS制御）

## 7. 運用性の工夫
- 記事データは`posts.js`でオブジェクト配列として管理し、JSON風の構造で追記可能にする。
- カテゴリやおすすめフラグなどメタ情報を持たせ、全ページで再利用。
- 未来の機能拡張用にフィルタ関数とユーティリティを`js/dataService.js`として分離。
